# Lexa_programming_language

Created with C++
The project includes a custom Lexer, Parser, Abstract Syntax Tree (AST), Symbol Table and Evaluator. Lexa was designed to provide a simple and readable syntax while exploring the core concepts behind programming language development.

- Variables and assignments
- Arithmetic expressions
- Comparison operators
- `if`, `else if`, `else`
- `while` loops
- `for` loops
- Block statements using `{ }`
- Increment and decrement operators (`++`, `--`)
- Compound assignments (`+=`, `-=`)
- Interactive REPL
- Custom AST-based execution
